
This is my high school project. It's e-shop made with HTML,CSS,PHP,MySQL and JavaScript. Blong and Forum were added later as additional features. Whole project has admin section where you can add new items for sale or new blogs posts and edit whole project. This project was made around April 2023

--HOW TO RUN--
Everything to run this project should be imported in this folder to just run it you will need to install XAMPP
or something similar and move this project into htdocs folder and also add eshopfd SQL export into your database.
